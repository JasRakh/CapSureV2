# Как подключиться через QR код

## Ваш IP адрес: 100.10.0.191

## Способ 1: Через QR код (обычный режим)

1. Запустите:
   ```bash
   ./start.sh
   ```
   Или:
   ```bash
   npx expo start --clear
   ```

2. В терминале появится QR код

3. Откройте **Expo Go** на телефоне

4. Нажмите **"Scan QR code"** и отсканируйте QR код из терминала

## Способ 2: Ручной ввод URL (если QR не работает)

1. В Expo Go нажмите **"Enter URL manually"**

2. Введите:
   ```
   exp://100.10.0.191:8081
   ```

3. Нажмите **"Connect"**

## Способ 3: Tunnel режим (работает везде)

```bash
npx expo start --tunnel --clear
```

Подождите 30-60 секунд пока установится tunnel, затем отсканируйте QR код.

## Важные проверки:

✅ **Телефон и компьютер должны быть в одной Wi-Fi сети** (для способов 1 и 2)
✅ **Expo Go должен быть установлен** на телефоне
✅ **В терминале не должно быть красных ошибок**
✅ **Порт 8081 должен быть свободен**

## Если ничего не работает:

1. Проверьте что в терминале есть строка:
   ```
   › Metro waiting on exp://100.10.0.191:8081
   ```

2. Попробуйте tunnel режим - он работает даже на разных сетях

3. Проверьте что Expo Go обновлен до последней версии

4. Перезапустите Expo Go на телефоне

5. Попробуйте перезагрузить Wi-Fi на обоих устройствах
