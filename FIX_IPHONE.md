# Исправление "пригодные данные не найдены" на iPhone

## Проблема
iPhone показывает "пригодные данные не найдены" при сканировании QR кода.

## Решение 1: Используйте Tunnel режим (РЕКОМЕНДУЕТСЯ)

```bash
npx expo start --tunnel --clear
```

Подождите 30-60 секунд пока установится tunnel, затем отсканируйте QR код снова.

## Решение 2: Проверьте что Metro Bundler работает

1. Запустите:
   ```bash
   npx expo start --clear
   ```

2. В терминале должна появиться строка:
   ```
   › Metro waiting on exp://...
   ```

3. Если видите ошибки (красный текст) - исправьте их сначала

## Решение 3: Тест с простым приложением

1. Переименуйте текущий App.tsx:
   ```bash
   mv App.tsx App.tsx.backup
   mv App-test.tsx App.tsx
   ```

2. Запустите:
   ```bash
   npx expo start --clear
   ```

3. Если простое приложение работает - проблема в коде основного приложения

4. Верните обратно:
   ```bash
   mv App.tsx App-test.tsx
   mv App.tsx.backup App.tsx
   ```

## Решение 4: Проверьте зависимости

```bash
rm -rf node_modules
npm install
npx expo start --clear
```

## Решение 5: Используйте симулятор iOS

Если у вас Mac:
```bash
npx expo start --ios
```

Это запустит приложение в симуляторе напрямую.

## Решение 6: Ручной ввод URL

1. В Expo Go нажмите "Enter URL manually"
2. Введите: `exp://100.10.0.191:8081`
3. Или используйте tunnel URL из терминала

## Что проверить в терминале:

После запуска `npx expo start` вы должны увидеть:
- ✅ QR код
- ✅ "Metro waiting on exp://..."
- ✅ Нет красных ошибок
- ✅ "Bundling JavaScript" когда загружается приложение

Если видите ошибки - пришлите их мне!
