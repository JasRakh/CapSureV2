# Конфигурация моделей OpenAI

## Доступные модели

### GPT-5 (если доступна)

- **gpt-5** или **gpt-5-turbo** - Самая новая и мощная модель
- Лучшее качество распознавания
- Выше стоимость

### GPT-4o (Рекомендуется)

- **gpt-4o** - Оптимизированная версия GPT-4 для vision задач
- Отличное качество распознавания изображений
- Хороший баланс цена/качество

### GPT-4o-mini (Быстрая и дешевая)

- **gpt-4o-mini** - Быстрая и экономичная альтернатива
- Хорошее качество при меньшей стоимости
- Рекомендуется для высокочастотного использования

### GPT-4 Turbo

- **gpt-4-turbo** - Стандартная GPT-4 Turbo
- Надежное качество

## Как изменить модель

### Вариант 1: Через переменную окружения (рекомендуется)

Создайте файл `.env` в корне проекта:

```
EXPO_PUBLIC_OPENAI_MODEL=gpt-5
```

Или для более дешевой модели:

```
EXPO_PUBLIC_OPENAI_MODEL=gpt-4o-mini
```

### Вариант 2: Прямо в коде

Откройте `services/openaiService.ts` и измените строку 24:

```typescript
const OPENAI_MODEL = process.env.EXPO_PUBLIC_OPENAI_MODEL || 'gpt-5'; // или 'gpt-4o', 'gpt-4o-mini'
```

## Автоматический Fallback

Приложение автоматически пробует другие модели, если основная недоступна:

1. Сначала пробует выбранную модель (например, gpt-5)
2. Если не работает - пробует gpt-4o-mini
3. Затем gpt-4-turbo
4. И наконец gpt-4

Это обеспечивает надежность работы даже если какая-то модель недоступна.

## Про "Unlimited" запросы

**Важно**: Полностью неограниченных запросов не существует. Все API имеют лимиты.

### Как увеличить лимиты:

1. **Upgrade ваш OpenAI план**

   - Зайдите на https://platform.openai.com/account/billing
   - Перейдите на более высокий тарифный план
   - Это увеличит ваши rate limits

2. **Добавьте кредиты**

   - Убедитесь, что у вас достаточно средств на счету
   - Добавьте кредиты через billing dashboard

3. **Используйте более дешевые модели**

   - GPT-4o-mini дешевле и имеет более высокие лимиты
   - Это позволит делать больше запросов

4. **Реализуйте кэширование**

   - Кэшируйте результаты для одинаковых изображений
   - Это уменьшит количество API запросов

5. **Используйте batch requests** (если доступно)
   - Некоторые модели поддерживают batch обработку
   - Это может быть более эффективно

### Текущие лимиты OpenAI (примерные):

- **Free tier**: ~3 запроса в минуту
- **Tier 1 ($5+ spent)**: ~60 запросов в минуту
- **Tier 2 ($50+ spent)**: ~500 запросов в минуту
- **Tier 3 ($500+ spent)**: ~5000 запросов в минуту

Проверьте ваши лимиты на: https://platform.openai.com/account/limits
