# Quick Start Guide

## Запуск приложения

### Вариант 1: Обычный режим (рекомендуется)
```bash
# Остановите все процессы на порту 8081
lsof -ti:8081 | xargs kill -9

# Запустите Expo
npx expo start --clear
```

Затем отсканируйте QR-код в Expo Go.

### Вариант 2: Tunnel режим (если обычный не работает)
```bash
# Сначала установите ngrok локально
npm install @expo/ngrok --save-dev

# Затем запустите с tunnel
npx expo start --tunnel --clear
```

### Вариант 3: Ручной ввод URL
Если QR-код не работает:
1. Найдите в терминале строку: `exp://192.168.x.x:8081`
2. В Expo Go нажмите "Enter URL manually"
3. Введите URL из терминала

## Устранение проблем

### Порт занят
```bash
# Убейте процесс на порту 8081
lsof -ti:8081 | xargs kill -9

# Или используйте другой порт
npx expo start --port 8082
```

### Tunnel не работает
Используйте обычный режим - он работает если телефон и компьютер в одной Wi-Fi сети.

### Приложение не загружается
1. Проверьте что все зависимости установлены: `npm install`
2. Очистите кэш: `npx expo start --clear`
3. Перезапустите Expo Go на телефоне
